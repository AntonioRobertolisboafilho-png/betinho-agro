# BETINHO AGRO v1.0

Aplicativo pessoal e gratuito para gestão da propriedade rural.

## Módulos
- Dashboard
- Suinocultura
- Avicultura
- Agricultura
- Estoque
- Financeiro
- Relatórios
- Configurações
- Assistente inteligente local

## Publicar gratuitamente no GitHub Pages
1. Crie um repositório público chamado `betinho-agro`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. No GitHub, abra Settings > Pages.
4. Em Source, selecione Deploy from a branch.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve e aguarde o endereço do aplicativo.

## Instalar no iPhone
1. Abra o endereço publicado no Safari.
2. Toque em Compartilhar.
3. Toque em Adicionar à Tela de Início.

## Dados
Os dados ficam salvos no navegador do aparelho. Use “Exportar backup” regularmente.
